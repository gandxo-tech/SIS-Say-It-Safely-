Politique de confidentialité – SIS

Dernière mise à jour : 07 août 2025

Bienvenue sur SIS – Say It Safely, un service d’envoi de messages anonymes. Votre vie privée est notre priorité.

1. Données collectées

  • Messages envoyés via le site
  • Adresse IP (via Firebase ou analytics)
  • Données techniques (navigateur, type d’appareil)

2. Où sont stockées vos données ?

Les messages sont stockés dans Firebase, un service sécurisé de base de données.

3. Utilisation des données

  • Affichage des messages sur votre profil
  • Statistiques (anonymes)
  • Lutte contre les abus (ex. : spam, propos haineux)

4. Partage

Aucune donnée personnelle n’est vendue ou partagée avec des tiers.
N'oubliez pas : ici, vous vous exprimez en toute sécurité.

5. Cookies & suivi

Des cookies peuvent être utilisés pour améliorer votre expérience (préférences, langue, etc.).

6. Sécurité

Firebase offre une sécurité avancée, avec authentification et règles d’accès strictes.

7. Suppression des données
